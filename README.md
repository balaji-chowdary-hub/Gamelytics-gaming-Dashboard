# Gamelytics-gaming-Dashboard
GAMELYTICS is a 4-page Power BI dashboard analyzing the gaming industry across revenue, platforms, player demographics, and reviews. It tracks $1.25B in revenue (2015–2024), player retention of 78.4%, platform &amp; genre performance, and sentiment analysis — delivering end-to-end insights through dynamic visuals and DAX-powered metrics.


##🎮 Gamelytics – Gaming Analytics Dashboard

## 📌 Project Overview

Gamelytics is an interactive Gaming Analytics Dashboard built using Power BI to analyze player behavior, revenue trends, game performance, and user engagement across multiple dimensions.

This dashboard helps stakeholders understand key metrics like churn rate, retention, revenue distribution, and player demographics to make data-driven decisions.

## 📂 Dataset Description

The dataset includes:

🎮 Game titles, genres, and platforms
💰 Revenue and units sold
👥 Player demographics (age, gender, country)
⭐ Review scores and sentiment
📅 Time-based data (year, month)
📊 Key Insights
📈 North America contributes the highest revenue (~27%)
🎯 RPG and Shooter genres dominate revenue share
📊 Peak revenue observed around 2022–2023
👥 Majority players belong to 18–34 age group
⭐ Average review score remains stable around 6.8–7.0
